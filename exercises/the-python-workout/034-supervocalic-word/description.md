# Exercise 34: Supervocalic Word

## Description

write a get_sv function that returns a set of all super- vocalic words in the dict. Simply put, such words contain all five vowels in English (a, e, i, o, and u), each of them appearing once and in alphabetical order. For the purposes of this exercise, we’ll loosen the definition, accepting any word that has all five vowels, in any order and any number of times. Your function should find all the words that match this definition (i.e., contain a, e, i, o, and u) and return a set containing them.
